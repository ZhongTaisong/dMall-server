## 技术栈

express + mysql + cors + multer


## 项目运行


```
git clone https://github.com/zhong-tsong/express-server-dmall  

cd express-server-dmall  

npm install 或 yarn

npm run start (需安装mysql，安装教程https://www.runoob.com/mysql/mysql-install.html)


用微信开发者工具访问

```
