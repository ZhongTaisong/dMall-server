## 技术栈

express + mysql + cors + multer


## 项目运行


```
git clone https://gitee.com/zhongts/nodejs-express-server-dmall2.0  

cd nodejs-express-server-dmall2.0  

npm install 或 yarn

npm run start (需安装mysql，安装教程https://www.runoob.com/mysql/mysql-install.html)


用微信开发者工具访问

```
